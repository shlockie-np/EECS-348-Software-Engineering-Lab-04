Files/Folders:
- nfl_scores.c
  NFL score combinations program (with input validation + Doxygen comments).

- temp_conv.c
  Temperature conversion utility (F/C/K), categorization/advisory, input validation + Doxygen comments.

- CMakeLists.txt
  Build configuration used by CLion/CMake to compile the C programs.

- Doxyfile
  Doxygen configuration file (controls how documentation is generated).

- docs/html/
  Generated Doxygen HTML documentation output.
  Open this file in a browser to view the docs:
    docs/html/index.html
  
Docs (Doxygen):
- To regenerate documentation, run in the project root:
    doxygen Doxyfile
- Then open:
    docs/html/index.html
