Detailed Description

Temperature conversion utility with categorization and advisory.

Features:

  • Converts between Fahrenheit, Celsius, Kelvin
  • Validates input (numeric parsing, choices, absolute zero constraint)
  • Categorizes temperature based on Celsius value and prints advisory text

◆ categorize_temperature()

void categorize_temperature ( float celsius)

Categorize temperature (based on Celsius) and print an advisory.

Categories (example thresholds):

  • c < 0 : Freezing
  • 0 <= c < 10 : Cold
  • 10 <= c < 25 : Comfortable
  • 25 <= c <= 35 : Hot
  • c > 35 : Extreme Heat

celsiusTemperature in Celsius used for categorization.
